# Momentum

Organize tasks. Fuel progress.

Momentum is a Progressive Web App (PWA) built to help users manage daily tasks, stay productive offline, and keep themselves motivated with personal notes.

## Features

- ✅ Create, edit, and delete tasks
- 💾 Persistent local storage with IndexedDB
- 📱 Installable as a PWA
- 📶 Works offline
- 📝 Personal motivation notes
- 🎯 Clean and responsive UI

## Tech Stack

- HTML
- CSS
- JavaScript
- IndexedDB
- Service Workers
- Web App Manifest

## Why Momentum?

Momentum is more than a todo app. It combines productivity with self-motivation, helping users stay consistent and focused even without internet access.

## Future Improvements

- Notifications/reminders
- Dark mode
